1.Install the python 3.11.9 
2. Verify Installation

Open Command Prompt and run:

py -0

Expected output should show both versions:

Installed Pythons:
 - 3.13
 - 3.11

3. Go to Your Project Folder

Example:

cd D:\Minor-Project\LLM-APP-WEB-SEARCH
4. Create a Virtual Environment Using Python 3.11
py -3.11 -m venv .venv

This creates a project-specific environment.

5. Activate the Environment
.venv\Scripts\activate

Your terminal should now show:

(.venv)

Verify Python version inside the environment:

python --version

Expected:

Python 3.11.9
6. Upgrade Pip
python -m pip install --upgrade pip setuptools wheel
7. Install Your Project Dependencies

Run:

pip install -r requirements.txt

Now packages like:

numpy

streamlit

chromadb

crawl4ai

should install without compilation errors.
